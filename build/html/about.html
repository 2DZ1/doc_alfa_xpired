Halaman Utama¶

Selamat datang! Disini berisi penjelasan dari Alfa Xpired.

Dimana Alfa Xpired adalah program berbasis website yang membantu pekerjaan kita dalam memanajemen tanggal expired produk. Apakah anda tau salah satu faktor penyebab NBH ditoko adalah barang expired? Untuk itu program ini tercipta.

Kenapa?¶

Berikut beberapa kelebihan ketika menggunakan Alfa Xpired.

✔ Data yang diinput akan lebih mudah dilihat
✔ Ketika barang sudah mendekati RH akan ada notifikasi lewat email
✔ NBH yang disebabkan oleh barang expired akan berkurang

Memulai¶

Sebelum menggunakan Alfa Xpired, Pastikan peramban web versi terbaru, untuk Internet Explorer tidak didukung di Alfa Xpired. Peramban web yang digunakan :

✔ Mozilla Firefox
✔ Google Chrome

Ketika anda sudah membuka browser Mozilla Firefox atau Google Chrome masukkan Alamat URL 10.234.94.113 selanjutnya tekan tombol ENTER. Tunggu hingga halaman utama terbuka. Ketika terbuka akan menampilkan halaman login.

Halaman Login Alfa Xpired

Ketika halaman ini sudah terbuka, akan menampilkan NIK dan PASSWORD untuk melanjutkan. Jika sudah memiliki akun di Alfa Xpired tinggal memasukkan NIK dan PASSWORD yang sudah dibuat. Jika anda belum memilikinya, bisa buat akun melalui COS , ACOS atau IT untuk dibuatkan akun tersebut.

Catatan

Jika anda sudah punya akun tapi tidak bisa masuk, atau lupa PASSWORD dapat menghubungi TEAM IT.
